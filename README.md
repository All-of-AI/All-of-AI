## All of AI

All of AI是一份由全国多个顶尖高校的多名研究生和本科生整理的人工智能学习**基础指南**。

目前，All of AI包含如下九个领域：

##### 1. 数理基础 Basic Mathematics (MATH)

该领域概括了学习AI所需的较为全面的数学知识，包括线性代数、微积分、数学优化、概率论、信息论、线性规划和谱图理论等。

##### 2. 机器学习 Machine Learning (ML)

该领域提供了主流的机器学习算法，包括线性回归、k近邻算法、贝叶斯分类器、支持向量机、决策树、集成学习、k均值算法、主成分分析、话题模型和概率图模型等。

##### 3. 深度学习 Deep Learning (DL)

该领域主要介绍各类神经网络模型，如全连接神经网络、卷积神经网络、循环神经网络、自编码器、深度生成模型、图神经网络等；并介绍了神经网络模型的各类学习形式与技巧，如神经网络的优化方式、注意力机制、小样本学习等。

##### 4. 自然语言处理 Natural Language Processing (NLP)

该领域专注于利用深度学习算法与思想解决自然语言处理领域的问题。从词向量和语言模型出发，逐个介绍了机器翻译、文本分类、文本摘要、机器阅读理解、对话系统等自然语言处理任务，并介绍了当前流行的预训练语言模型，如BERT、ERNIE、GPT-2等。

##### 6. 信息检索 Information Retrieval (IR)

该领域包含信息检索的基础理论，同时还有近年来发展的IR子领域，如排序学习、个性化与多样化检索、澄清式检索等。
